# 📖 推送代码到 GitHub - 详细步骤指南

## 方法一：使用 GitHub Desktop（最简单，推荐）

### 第一步：下载 GitHub Desktop

1. 打开浏览器
2. 访问这个网址：https://desktop.github.com/
3. 点击绿色的 "Download for macOS" 按钮
4. 下载完成后，双击安装文件
5. 按照提示安装（一直点"继续"或"下一步"）

### 第二步：登录 GitHub

1. 打开 GitHub Desktop 软件
2. 点击 "Sign in to GitHub.com"
3. 输入你的 GitHub 账号和密码登录

### 第三步：添加你的项目

1. 在 GitHub Desktop 左上角，点击 "File"（文件）
2. 选择 "Add Local Repository"（添加本地仓库）
3. 点击 "Choose..."（选择）
4. 找到并选择这个文件夹：`/Users/liuyujia/Downloads/ai-resume-builder (1)`
5. 点击 "Add repository"（添加仓库）

### 第四步：推送代码

1. 在 GitHub Desktop 左下角，你会看到 "2 commits ahead of origin/main"
2. 点击右上角的 "Publish branch"（发布分支）按钮
3. 等待上传完成（会显示进度条）
4. 完成后，你的代码就上传到 GitHub 了！

## 方法二：使用终端（Terminal）

### 第一步：打开终端

1. 按键盘上的 `Command（⌘）` + `空格键`
2. 输入 "终端" 或 "Terminal"
3. 点击 "终端" 应用程序

### 第二步：进入项目文件夹

1. 在终端窗口中，复制粘贴下面这行命令：
   ```
   cd "/Users/liuyujia/Downloads/ai-resume-builder (1)"
   ```
2. 按键盘上的 `回车键（Enter）`

### 第三步：推送代码

1. 复制粘贴下面这行命令：
   ```
   git push origin main
   ```
2. 按 `回车键（Enter）`
3. 如果要求输入用户名，输入：`q755015108`
4. 如果要求输入密码，输入你的 GitHub 密码（或者 Personal Access Token）

### 如果提示需要密码（Token）

如果 GitHub 要求输入密码，你需要：

1. 打开浏览器，访问：https://github.com/settings/tokens
2. 点击 "Generate new token" → "Generate new token (classic)"
3. Token name（令牌名称）填写：`resume_editor`
4. 勾选 "repo" 选项
5. 点击 "Generate token"（生成令牌）
6. 复制生成的令牌（一串很长的字符）
7. 在终端输入密码时，粘贴这个令牌（而不是你的 GitHub 密码）

## 方法三：直接在 GitHub 网页上传（最简单但需要重新上传）

如果上面两种方法都太复杂，你可以：

1. 访问：https://github.com/q755015108/resume_editor
2. 点击 "uploading an existing file"（上传现有文件）
3. 把项目文件夹里的所有文件拖拽上传
4. 点击 "Commit changes"（提交更改）
